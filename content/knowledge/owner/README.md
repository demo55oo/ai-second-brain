# Owner brain (runtime only)

This folder does **not** ship your personal markdown in git.

- **Local:** uploads merge into `BRAIN.md` here (gitignored).
- **Vercel:** uploads merge into Blob at `owner/BRAIN.md` (token auto-injected by Deploy button).
- Uploaded `.md` / `.zip` files are **not** kept as separate files — only sections inside that one brain doc.
- Bundled demo knowledge stays under `content/knowledge/danny/` until you upload.
